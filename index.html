Proyecto ToDoList

Este proyecto consiste en una aplicación de tareas (ToDoList) con arquitectura modular y enfoque en seguridad, validación visual.
El backend, desarrollado con Flask y desplegado en Railway, ofrece autenticación con JWT, registro de usuarios y operaciones CRUD sobre tareas, integrando Supabase como base de datos.
El frontend, alojado en GitHub Pages, simula flujos reales de usuario, valida endpoints en tiempo real y presenta una interfaz clara con formularios funcionales, animaciones GSAP y componentes visuales interactivos.
